# uinve — 체육대학 입시 상담 자료

체육계열 대학 입시 상담 도구를 위한 **데이터 자산 + 실기 채점 엔진 + 검색 프로토타입**.

학생의 내신·모의고사·실기 기록을 입력해 대학별 환산 총점을 계산하고, 학과계열·전형·지역·실기종목으로 검색하는 상담 도구를 목표로 한다. 기존 앱([today119/sports-college-admission](https://github.com/today119/sports-college-admission))의 환산 엔진·상담기록 구조에, 더 충실한 2028 전국 데이터와 PDF 실기 채점표를 결합하기 위한 작업 저장소.

## 구성

```
uinve/
├─ app/                         검색/카드 프로토타입 (Vite + React)
│  └─ src/data/                 데이터 자산 (아래)
│     ├─ admissions.json        2028계획안 추출: 수시 477 + 정시 247 모집단위
│     ├─ silgi_tables.json      PDF 채점표 원추출 (22개 대학)
│     ├─ silgi_scoring.json     정규화 채점데이터 (임계값식, 22개 대학)
│     ├─ silgi_scoring_clean.json  종목명 정상 = 즉시 사용가능 (10개 대학)
│     └─ silgiScore.js          앱용 실기 채점 함수
└─ scripts/                     데이터 생성 파이프라인
   ├─ 1_extract_silgi.py        PDF → 채점표 추출 (목차맵 기반)
   ├─ 2_normalize_silgi.py      채점표 → 임계값 정규화 + 검증
   └─ toc_map.json              목차: 95개 대학 → 시작페이지
```

## 데이터 출처
- **2028계획안.xlsx** — 마스터 DB. 전국 18개 지역, 수시 118대학·정시 126대학. 반영비율·등급환산·수능영역반영·실기종목.
- **2027학년도 대입 체육계열 실기전형 자료집.pdf** (311p) — 대학별 종목별 기록→점수 채점표. 실기 자동계산의 유일한 출처.

## 실기 채점 엔진

PDF 채점표를 임계값 형태로 정규화해, 측정 기록을 점수로 환산한다.

```js
import { silgiScore } from './app/src/data/silgiScore.js'
silgiScore('고려대학교', '남', '메디신볼', 8.2)  // → { ok:true, score:72 }
```

- `dir:'higher'` — 기록이 클수록 고득점(멀리뛰기·메디신볼·윗몸일으키기)
- `dir:'lower'` — 기록이 작을수록 고득점(달리기류)
- 고려대 채점표로 7/7 검증 완료.

## 진행 상태
- ✅ 2028 전국 데이터 추출 (수시 477 + 정시 247)
- ✅ 검색/카드 프로토타입 (전형유형·학과계열·실기비중·지역·실기종목 필터)
- ✅ 실기 채점 엔진 + 즉시사용 10개 대학
- ⬜ 채점표 라벨 보정 12개 대학 / 변형 포맷 73개 대학 확장
- ⬜ 기존 앱 `silgiEvents`에 채점데이터 주입 (실기→총점 관통)
- ⬜ 합격컷 데이터 별도 확보
